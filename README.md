# Pointbreak Debug Documentation

This repository is the historical source for the retired Pointbreak Debug documentation. Its
Mintlify configuration, instructional and legal pages, snippets, and publication assets have been
removed from the default branch.

Pointbreak Debug is no longer active.
This repository is not current product documentation or a legal or support authority. For current
work, use
[Pointbreak](https://github.com/withpointbreak/pointbreak), the review record for understanding what
a coding agent changed and why.

The removed documentation remains available in Git history. The historical source remains licensed
under [CC BY 4.0](LICENSE).
